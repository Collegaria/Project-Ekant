* Environment activation
- Remember to install ~virtualenv~ with

  ~python3 -m pip install --user virtualenv~

- On Windows, run:
  ~env\Scripts\activate.bat~

- On Unix or MacOS, run:
  ~source env/bin/activate~

- Run ~deactivate~ to deactivate
